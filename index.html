Monty Hall Probability

The program calculates the probability of the winning strategy between stick and switch. The minimum number of doors is 3. The difference between the total number of doors and the number of doors the host can open should be at least 2, to simulate probability between 2 doors. The program assumes that 1 of the doors is the user's choice.
This table is an example of possible input values for the total number of doors and min/max values of the doors the host can open.
Total doors Host can open
3 ( 1 )
4 ( 1 | 2 )
5 ( 1 | 2 | 3 )
6 ( 1 | 2 | 3 | 4 )
7 ( 1 | 2 | 3 | 4 | 5 )
